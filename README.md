#  Stock Portfolio Tracker

A simple Python application that calculates the total investment value of stocks using predefined stock prices. Users can enter stock names and quantities, and the portfolio summary is automatically saved to a text file.

##  Features
- Stock price tracking using dictionaries
- User input for stock selection and quantity
- Investment value calculation
- Portfolio summary generation
- File handling with text file storage

##  Technologies
- Python
- Dictionaries
- File Handling
- Conditional Statements
